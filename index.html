Приложение XY Glove — Ваш помощник в управлении перчаткой

Приложение XY Glove позволяет пользователям контролировать работу перчатки и персонализировать упражнения в зависимости от уровня моторики ребенка. Оно создано для того, чтобы сделать процесс реабилитации простым, увлекательным и адаптированным к каждому пользователю.

Основные функции приложения:

  • Управление стимуляцией: Через приложение можно настраивать интенсивность импульсов для перчатки.
  • Цветовой интерфейс: Простой и удобный цветовой круг для легкого выбора настроек.
  • Упражнения по уровням: Программа тренировок для разных уровней моторики.
  • Интерактивное рисование: Позволяет детям развивать креативные способности.
  • Отслеживание прогресса: Данные о выполненных упражнениях и результатах.
